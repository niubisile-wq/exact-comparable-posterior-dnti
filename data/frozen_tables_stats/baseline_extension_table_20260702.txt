Interpretation
- The fixed-load-factor DNN baseline is far below the variable-load-factor NRE
  model on both systems, which supports the value of load-aware posterior
  modeling rather than a fixed-condition point classifier.
- GraphSAGE is competitive on 33-bus but weaker on 69-bus; it remains a
  point-estimate baseline rather than a posterior method.
- For the enumerable 33-bus and 69-bus libraries, the exact-enumeration
  posterior is the correct reference label. An AIS-style Bayesian computation
  over the same finite library reaches the same top-1 upper bound, so the
  manuscript keeps the exact-enumeration label to avoid overstating a Xu-style
  AIS comparison where enumeration is already available.
