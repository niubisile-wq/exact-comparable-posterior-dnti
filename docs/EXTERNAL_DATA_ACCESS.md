# External Data Access Notes

This manuscript uses public or third-party benchmark/network/measurement sources as realism anchors and controlled stress cases. Raw third-party data should not be redistributed unless the original license or access terms allow redistribution.

## SoCal Public-Measurement Replay

The public release should include:

- the source URL or formal access citation;
- the date accessed;
- the exact filtering and blocked cross-validation protocol;
- the topology-state count and support per state;
- a statement that no timestamp feature was used in the reported blocked-CV classifier.

## Benchmark Network and Three-Phase Assets

For IEEE-style or public benchmark feeders, include formal citations and access paths rather than bundling restricted raw assets. Derived result manifests and figure source data can be shared if they do not violate source terms.

## Public Release Rule

If redistribution rights are unclear, publish:

- access instructions;
- preprocessing scripts;
- derived non-sensitive result manifests;
- figure source data;
- checksums or filenames for reproducibility.

Do not publish restricted raw third-party files.
